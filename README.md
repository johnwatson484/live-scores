# Live scores publisher
Publish live scores

## Prerequisites

Either:
- Docker
- Docker Compose

Or:
- Node 18+

## Running application
### Docker
```
docker-compose build
docker-compose up
```

### Node
```
node app
```
Or:
```
nodemon app
```
